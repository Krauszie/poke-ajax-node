<h1>Poke API with jQuery & Node.js</h1>
<h2>Installation</h2>
<ul>
  <li>Clone Repo</li>
  <li>npm install -g http-server</li>
  <li>npm install</li>
</ul>

<h2>Tech Stacks</h2>
<ul>
  <li>Ajax/jQuery</li>
  <li>TailwindCSS</li>
  <li>Node.js</li>
  <li>Express</li>
  <li>PostgreSQL</li>
</ul>

<h2>How To Run</h2>
<ul>
  <li>npm start the server</li>
  <li>http-server -p <port> --cors</li>
</ul>
